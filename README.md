# Loan From Home 💰🏠

A modern, responsive single-page website for a financial loan company called **"Loan From Home"**.

The website allows users to easily explore various loan services such as Personal Loans, Home Equity Loans, Student Loans, Mortgage Loans, and Payday Loans — all from the comfort of their home.

### ✨ Features

- Fully responsive design (Mobile + Tablet + Desktop)
- Modern and clean UI using Bootstrap 5
- Four main sections in the footer with Services, About, How it Works, and Pages
- Attractive Subscribe section with email input
- Social media icons and branding (Wallet logo)
- Smooth and professional look suitable for fintech projects

### 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome / Bootstrap Icons
